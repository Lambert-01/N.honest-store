# N.Honest Supermarket

![N.Honest Logo](images/f.logo.png)

N.Honest Supermarket is an online grocery store that brings you the finest selection of fresh produce, household essentials, and lifestyle products—all under one roof. We prioritize quality, affordability, and customer satisfaction to ensure a seamless shopping experience.

## Features

- **User-Friendly Interface**: A clean and intuitive design for easy navigation.
- **Dynamic Product Management**: Add, edit, and delete products via the admin panel.
- **Category Filtering**: Browse products by category for a personalized shopping experience.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
- **Secure Backend**: Built with Node.js, Express, and MongoDB for reliable data management.

---

## Technologies Used

- **Frontend**:
  - HTML5, CSS3, JavaScript (ES6+)
  - Bootstrap 5 for responsive design
  - Swiper.js for carousels and sliders
- **Backend**:
  - Node.js and Express.js
  - MongoDB (with Mongoose ODM)
  - Multer for image uploads
- **Tools**:
  - Git for version control
  - npm for dependency management
